Ohjelmointikieliä

C++

C++ on monipuolinen ohjelmointikieli jota käytetään ensisijaisesti pelien ohjelmointiin, vaikea oppia mutta sen arvoinen. kehitettiin 1980-luvulla C kielestä paremmaksi.

JavaScript

JavaScript on tärkeimmillään skriptikieli web-sivujen ohjelmointiin, käyttökokemuksen ja käyttöliittymän toiminnallisuuden parantamiseen.

C#

C# on myös peliohjelmointikieli, mutta se on paljon erilaisempi kuin C++ kuin C# on syntaksiltaan lähellä php:tä eikä muistuta C:aa juuri lainkaan.

PHP

PHP, eli hypertekstin esikäsittelijä jolla voi toteuttaa web-pohjaisia ohjelmointiratkaisuja.Sitä voi yhdistää HTML-koodin kotisivuihin ja sen kanssa toimii paljon tietokantoja esim. MySql.

CSS

CSS on kieli joka määrittää verkkosivujen ulkonäköä, erottuu JavaScriptistä. JavaScript on enemmän käytetty verkkosivujen toimintaan kuin ulkonäköön, CSS on pelkästään ulkonäkö kieli. Helppo ymmärtää, kuin sitä käytetään jokaisessa sivustossa.
